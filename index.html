Startle

How to play

Everyone has their starting word.. including the computer! The same starting word was used for each game displayed. Use the 5 completed game results to determine what the starting word was.

You have an unlimited number of tries to guess the word. Try unchecking the answer feedbox option to make it a bit tougher!

Hint: The computer is pretty efficient at guessing new words. This means that:

  • If the letter was green, future guesses will always repeat that letter in that location.
  • If the letter was yellow, future guesses will always include that letter, but not in that same spot.
  • If the letter was black, future guesses will not include that letter.
